# Finance Tracker Dashboard

A full-stack **Personal Finance Tracker Web App** built with **React, Tailwind CSS, Node.js, Express, and MongoDB**.  
This app lets users register, login, track incomes and expenses, view dashboard stats, and interact with an AI chatbot.
